Usage

Dependencies

This plugin has a few dependencies which are included. To set them up place the following in the <head> ... </head> section of your page: 
<link rel="stylesheet" type="text/css" href="style/ionicons.min.css" />
<link rel="stylesheet" type="text/css" href="style/player.css" />
<link rel="stylesheet" type="text/css" href="style/perfect-scrollbar.min.css" />
<script type="text/javascript" src="https://cdnjs.cloudflare.com/ajax/libs/jquery/2.1.4/jquery.min.js"></script>
<script type="text/javascript" src="js/soundmanager2.js"></script>
<script type="text/javascript" src="js/waveform.js"></script>
<script type="text/javascript" src="js/perfect-scrollbar.jquery.min.js"></script>
<script type="text/javascript" src="js/scplayer.js"></script>

Player Element

The player element can be any block element in the <body> ... </body> section of your page. You may assign the width attribute to size the player within the player element. Setting width="100%" or simply not including the tag sets the player to scale with the element.

Example: 
<div id="soundcloudPlayer" width="800"></div>

Initalizing

scPlayer.init(string url/artist, element, bool autoplay)
var player = new scPlayer();
player.init('http://www.soundcloud.com/trewth1', $('#soundcloudFrame'), true);
